# Vendored icon licenses

本ディレクトリの SVG は以下の OSS アイコンセットから**アイコン単位で** vendored している
(spec §12 — 主 Lucide・補 Tabler)。SVG は無改変 (各ファイル先頭の @license ヘッダ参照)。
着色は実行時に `currentColor` をテーマトークンへ置換して行う (theme/icons.py)。
