# .kiro/ — アーカイブ

- **新規の計画は `docs/superpowers/specs/`（設計）・`docs/superpowers/plans/`（実装プラン）** で行う（superpowers 駆動）。
- 旧 `.kiro/steering/` のルールは `docs/`（product / development / structure / policies / workflow）へ移設済み。
- 本ディレクトリの内容は編集しない（アーカイブ）。
